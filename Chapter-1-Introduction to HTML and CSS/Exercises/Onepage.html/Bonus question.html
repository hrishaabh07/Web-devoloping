HTML & CSS Quick Reference Guide

A handy guide for common HTML tags and CSS properties

HTML Tags

<h1> to <h6>

Used for headings from largest (<h1>) to smallest (<h6>).

<h1>Main Heading</h1>
<h2>Subheading</h2>

<p>

Paragraph tag for blocks of text.

<p>This is a paragraph.</p>

<a>

Defines a hyperlink.

<a href="https://example.com">Visit</a>

<img>

Embeds an image.

<img src="image.jpg" alt="Description" />

<ul>, <ol>, <li>

Lists: unordered, ordered, and list items.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<div>

Generic container for grouping content.

<div class="container">Content</div>

<span>

Inline container used for styling part of the text.

<p>Click <span class="highlight">here</span>.</p>

<form>, <input>, <button>

Creates interactive forms.

<form>
  <input type="text" placeholder="Your name" />
  <button>Submit</button>
</form>

<code>, <pre>

Used for displaying formatted code blocks.

<pre>
  <code>console.log("Hello World");</code>
</pre>
